# 🎵 Rock That Body Song(TIKTOK trend) - Lyrics Animation 🌈

A Python terminal project that displays animated song lyrics with rainbow colors and typewriter effects.

## ✨ Features
- Rainbow color animation 🌈  
- Typewriter-style lyric display ⌨️  
- Smooth timing effects 🎵  
- Terminal-based visual experience 💻  

## 🛠️ Tech Stack
- Python  
- Rich library  
- Time module  

## 🚀 How to Run
```bash
git clone https://github.com/kasuni-17/Rock-That-Body-trendy-song-lyrics.git
cd Rock-That-Body-trendy-song-lyrics
pip install rich
python lyrics.py
